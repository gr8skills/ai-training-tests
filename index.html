⚖️ Realistic rating tasks (the actual work)

This is the format that platforms like Outlier actually use — including multi-turn conversation tasks. For each task you read a prompt (or a full chat history) and two AI responses, score each on rubric dimensions, pick an overall preference on a 7-point scale, and write a justification — then an expert key shows the consensus rating and a model rationale to compare against.

📖 Rating guidelines (read me)
  • Instruction Following — did the response do exactly what was asked, including format, length, and audience constraints?
  • Truthfulness / Accuracy — are all claims correct? Penalize hallucinations and reasoning/calculation errors heavily.
  • Harmlessness — is it safe? Watch for both harmful over-helpfulness and unnecessary over-refusal of benign requests.
  • Quality — clarity, structure, and usefulness for the intended reader.
  • Context / Consistency (multi-turn tasks) — did the reply use everything established earlier in the conversation: constraints the user gave, things already tried, and the tone/level set in prior turns?
  • Score each response 1 (poor) – 5 (excellent) per dimension, then set an overall preference. Your justification must cite specific, concrete differences, not "A is just better".
